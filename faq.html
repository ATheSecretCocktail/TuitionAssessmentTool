Do I need to input values for every field?

You must enter values for all fields without a checkbox, as well as for any optional cost that you check off. The tool will notify you if you have an invalid inputs when you try to leave a page.

I'm not sure what value to input for a certain field. What should I do?

You can use the Fill Empty button to fill fields you're unsure of with default values. Note: these default values are only estimates to help you.

How do I get to a certain page?

You can use the tabs at the top of the page or the buttons at the bottom to move between pages.

How can I get a copy of my results?

You can download a copy of your report, including your input costs and tuition estimates, at the end of the tool.

Can I get a visualization of my results?

Yes, a visualization of your results will be provided at the end of the tool once all values are inputted.
